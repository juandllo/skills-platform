# Bienvenido a la Plataforma MD

Esta es una aplicación de escritorio construida con **Python** y **PySide6** para visualizar tus archivos Markdown de forma elegante.

## Características
- **Renderizado Rápido:** Visualización instantánea.
- **Estilo Limpio:** Basado en el diseño de GitHub.
- **Navegación Intuitiva:** Selecciona archivos desde el sidebar.

    print(f"Hola, {nombre}!")
